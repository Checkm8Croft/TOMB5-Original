# TOMB5 porting on OpenGL
Project of porting TOMB5 from Windows and DirectX to macOS/Linux and OpenGL

## Building
First make the build directory: `mkdir build && cd build`
For build the Emulator, just type: `cmake ..`
For build PC Version: `cmake ../SPEC_PC_N`
